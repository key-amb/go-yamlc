# yamlc

A YAML syntax checker.

```bash
# Check YAML syntax
yamlc [--[s]ilent] foo.yml
cat foo.yml | yamlc [--[s]ilent]

# Show version
yamlc --[v]ersion

# Show help
yamlc --[h]elp
```

# Install

```bash
go get github.com/key-amb/yamlc
```

Or get binaries at [Releases](https://github.com/key-amb/yamlc/releases) page.
